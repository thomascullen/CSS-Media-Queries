CSS-Media-Queries
=================

##iPhone
```css
@media only screen 
and (min-device-width : 320px) 
and (max-device-width : 480px) {
/* Styles */
}
```

##iPhone portrait
```css
@media only screen 
and (min-device-width : 320px) 
and (max-device-width : 480px) 
and (orientation : portrait) {
/* Styles */
}
```

##iPhone landscape
```css
@media only screen 
and (min-device-width : 320px) 
and (max-device-width : 480px) 
and (orientation : landscape) {
/* Styles */
}
```
